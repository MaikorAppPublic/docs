Byte Order

Each instruction must be written in the following order:

OP ARG1 ARG2.. ARG1_OFFSET ARG2_OFFSET..

For example:

InstructionOrderBytes
INC.B ALOP ARG12C 01
ADD.W $200 (AX+10)OP ARG1 ARG2 ARG2_OFFSET27 00 C8 B9 00 0A
